# Calculator

Calculator app.

## Instructions

To start this app:

- Installing Dependencies:

```
npm ci
```

- Compiles and hot-reloads for development:

```
npm run start
```

- Compiles and minifies for production:

```
npm run build
```

- Run tests:

```
npm run test
```

- Run lint:

```
npm run lint
```

- Format files:

```
npm run fmt
```
